# 🚀 space-shooter
A 2D arcade space-shooter game built using [Godot](https://godotengine.org/). This project was created by following a tutorial series by Heartbeast.

## 🎮 Gameplay
- Try to survive as much as possible to flex your highscore with friends (if you have them)
- Controls: Arrow Keys

## ❓ How to Play
1. Download the `.zip` archive you need based on your operating system from [Releases](https://github.com/shizuokalol/space-shooter/releases). This game natively exported to Windows x64 and Linux x64.
2. Extract executable file from the `.zip` archive and launch it. **NOTE**: On Linux, you may need to allow the file to be executed.
